# Personal Portfolio Website

This repository contains the source code for my **personal portfolio website**.  
It is built to showcase my projects, technical skills, and academic background as a **Computer Science Engineering undergraduate (CSE ’26)**.

The portfolio reflects my learning journey, hands-on project work, and interest in building clean, functional, and user-friendly web applications.

---

## About Me

I am a Computer Science Engineering student with a strong interest in software development and problem solving.  
I enjoy working on real-world projects, learning modern web technologies, and improving my development skills through practice and experimentation.

This portfolio is a single place to explore my work, understand my technical stack, and get in touch with me.

---

## What This Portfolio Includes

- Introduction and brief background
- Technical skills and tools I work with
- Selected projects with descriptions
- Education details
- Resume download option
- Contact and social links

The focus is on **clarity, simplicity, and professionalism**.

---

## Tech Stack Used

This project is built using modern and reliable frontend technologies:

- **React**
- **Vite**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**

These tools help in building a fast, responsive, and maintainable application.

---

## Project Structure (Overview)

- `src/` – Main application source code
- `components/` – Reusable UI components
- `pages/` or `sections/` – Portfolio sections (About, Skills, Projects, etc.)
- `assets/` – Images and static resources
- `styles/` – Global and component-level styles

---

## Running the Project Locally

To run this project on your local system, make sure **Node.js** and **npm** are installed.

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate into the project directory
cd <YOUR_PROJECT_NAME>

# Install all dependencies
npm install

# Start the development server
npm run dev
